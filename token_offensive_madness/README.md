- **Size**
  - nametag: 170 x 170px
  - no-nametag: 140 x 140px
- **Color**

- **Archetypes**
  - Spirit, Blade, Nightmare
  - Knight, Maiden, Chariot
  - Lord, Legion
  - Kingdom
  - Daemon
